# Contributing

PhishGuard Lite accepts defensive cybersecurity improvements only.

## Good Contributions

- New phishing detection rules
- Better report output
- More sample emails with fake data
- More tests
- Documentation improvements

## Not Accepted

- Credential collection
- Phishing kit code
- Email sending automation
- Website exploitation
- Instructions for bypassing security controls

## Development

Run tests before sharing changes:

```powershell
npm test
```
